# Calculadora Simples

> Projeto: **Calculadora Simples** — HTML, CSS e JavaScript

---

## Descrição

Este é um projeto pequeno e didático que implementa uma **calculadora básica** (adição, subtração, multiplicação e divisão) usando **HTML**, **CSS** e **JavaScript** puros. Ideal para demonstrar conhecimentos fundamentais de front-end: estruturação, estilo e manipulação do DOM.

---

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)

---

## Recursos / Funcionalidades

- Entrada de dois números
- Seleção da operação por *radio buttons* (soma, subtração, multiplicação, divisão)
- Validação implícita ao converter entradas para números (uso de `Number()`)
- Resultado exibido dinamicamente na página
- Estilo responsivo simples e tema dark (customizável)

---

## Como executar localmente

1. Clone o repositório:

```bash
git clone https://github.com/caires-tech/Calculadora-simples.git
```

2. Entre na pasta do projeto:

```bash
cd Calculadora-simples
```

3. Abra o arquivo `index.html` no seu navegador (duplo clique ou via Live Server no VS Code):

- Duplo clique no `index.html` ou
- Se usar VS Code, instale a extensão **Live Server** e clique em *Go Live* para rodar localmente.
